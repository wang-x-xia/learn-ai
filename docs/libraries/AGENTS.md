## 写法指南

- **属性表 + 一句话定位**：保留
- **技术亮点**：聚焦差异化实现，说明"为什么值得知道"而非"怎么用"
- 不写教程代码，不搬运 README
- 被其他文档频繁引用的库，在相关文档中有交叉引用
