# Trae

> 字节跳动推出的免费 AI IDE，技术上是标准的 VS Code fork + 前沿模型接入，核心差异在于商业策略。

| 属性 | 值 |
|------|-----|
| 厂商 | 字节跳动 (ByteDance) |
| 形态 | 桌面 IDE（macOS / Windows / Linux） |
| 开源 | 否（免费使用） |
| 技术栈 | Electron, TypeScript |
| 底座模型 | Claude Sonnet, GPT-4o（免费，字节代付） |
| 官网 | [trae.ai](https://trae.ai/) |

## 技术亮点

坦率地说，Trae 当前缺乏显著的技术差异化。其补全、对话、Agent 能力均为 AI IDE 的标配功能，无自研模型或独特架构。

值得关注的是商业层面：

- **免费前沿模型策略**：字节全额补贴 Claude / GPT 使用费用。这不是技术创新，而是一种用户获取策略——将 AI 编程工具作为流量入口，用烧钱换市场份额。
- **面向海外市场**：非中国大陆特供版，是字节全球化布局的一部分。

## 定价

- 完全免费（包括前沿模型使用）
